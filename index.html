The field of web development has evolved significantly, transitioning from static information websites to dynamic and interactive platforms driven by user-generated content. This shift has not only created new business opportunities but also transformed the way people interact with each other online. In this course, CITS3403/5505 Agile Web Development, students will delve into the technologies, programming languages, and environments that form the backbone of modern Internet development. The course covers the fundamentals of the web and the Internet, exploring key browser technologies such as HTML, CSS, and JavaScript. Students will learn about the development process for building large-scale web applications and gain hands-on experience with full-stack web development using technologies like Flask, AJAX, and SQLite. Overall, this course aims to equip students with the necessary skills to understand and leverage the technologies that power the modern web. By blending theoretical knowledge with practical application, students will be able to develop and showcase their abilities in building dynamic, interactive, and agile web solutions.

  • Explain the key components that enable the delivery of data and services on today's internet.
  • Understand the technologies behind the terms and how they fit together, with hands-on programming experience.
  • Develop web applications using client-side and server-side technologies.
  • Explain and apply agile project methodologies.
  • Design and implement secure web applications using industry best practices.
  • Collaborate effectively in web development teams using agile project methodologies.
